    • Features

      • Remote control camera from web browsers.
        • Mirroring camera screen
        • Remote control functions
      • Silent shutter on/off (NX500 only)
      • LCD on/off

    How To Install

    Camera

    • Prerequisites

      • NX500 with firmware version 1.12 or NX1 with firmware version 1.41
      • nx-wake-on 1.11
      • (optional) nx-patch 5.3
    1. Download https://github.com/mewlips/nx-remote-controller-mod/raw/master/nx-remote-controller-mod-v0.7.zip
    2. Unzip the nx-remote-controller-mod-v0.7.zip file to root of your SD card.
    3. Insert the SD card into your camera and power up the camera.
    4. Wait. The installation will be completed soon.

    Android

    • Install the https://github.com/mewlips/nx-remote-controller-mod/raw/master/NXRemoteController-v0.7.apk

    How To Use

    1. Press the 'MOBILE' button.
      • Basic 'MOBILE' function replaced by nx-remote-controller-mod.
      • It will open the Wi-Fi Settings app if the camera is not connected to a Wi-Fi network.
      • If you have a configured Wi-Fi network, the camera will be automatically try to connect to the configured Wi-Fi AP.
      • After the camera is connected on network, then app menu will be launched.
    2. Make sure the camera and the PC or Mobile device are on the same Wi-Fi network.
    3. Find IP Address of your camera from the app menu.
    4. Open 'http://' in web browser.
